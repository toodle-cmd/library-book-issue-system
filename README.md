# Library Book Issue System

This is a beginner-friendly Python project that simulates a library book issuing system.

## Features
- Student ID validation
- Book availability checking
- Author and book verification
- Automatic return date calculation
- Book reissue option
- Library guidelines display

## Technologies Used
- Python
- Dictionaries
- Conditional Statements
- DateTime Module

## How It Works
1. Enter student details.
2. Select a book and author.
3. The system checks whether the book is available.
4. If available, the book is issued and a return date is generated.
5. Users can reissue the book to extend the return date.
